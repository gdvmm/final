Mis Trabajos

HTML

Hypertext Markup Language o el lenguaje de marcas hipertexto, con el que se construyen las páginas web. Mi primer acercamiento a la programación, donde pude conocer la estructura básica de un archivo HTML, incluyendo doctype, html, head y body . Etiquetas tales como: títulos, subtítulo, párrafos. Agregar listas, imágenes y links.

CSS

CSS (siglas en inglés de Cascading Style Sheets), en español «Hojas de estilo en cascada».
Apendi a colocar una imagen de fondo y un texto encima , también imágenes con sus textos descriptivos de acompañamiento. Cargar el CSS de un archivo CSS externo, desde una subcarpeta css dentro de la subcarpeta assets.

FLEXBOX

Diseño de Caja Flexible, es un modelo de diseño CSS.En este caso ubique horizontalmente los elementos: Sección, párrafo e imagen. Agregó iconos con Font Awesome. Y a su vez utilicé etiquetas semánticas para definir las distintas secciones de la página.

Bootstrap

Bootstrap es una biblioteca multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web. Contiene plantillas de diseño con tipografía, formularios, botones, cuadros, menús de navegación.En este trabajo utilice Clases para margin y padding y diseñar las Cards.
